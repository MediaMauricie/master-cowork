🎨 Colorisation de base — pourquoi c'est nécessaire
Les fichiers sont remis en format brut natif — c'est le format tel qu'enregistré par la caméra, sans traitement de couleur appliqué. Les images apparaissent délavées et plates jusqu'à colorisation. La colorisation de base couvre les ajustements nécessaires pour une utilisation directe : application d'un LUT, balance des blancs, cohérence visuelle entre les clips et correction d'exposition. Pour les projets multicaméras, les effets avancés ou l'étalonnage cinématographique, une soumission séparée sera fournie.
